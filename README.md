# Simple TypeScript

Simple TypeScript starter project.

## Scripts

```bash
# Install dependencies
$ npm install

# Start application
$ npm start

# Build
npm run build

# Lint
$ npm run lint
$ npm run lint:fix

# Test
$ npm run test
$ npm run test:watch
$ npm run test:cov

# Clean
$ npm run clean                   # Removes dist folder
$ npm run clean:packages          # Removes node_modules folder
$ npm run clean:all               # Runs all of the above
```
